CEF OAuth Test Application

About This Application

This is a test application built with the Chromium Embedded Framework (CEF) that demonstrates OAuth integration. Click the buttons below to test OAuth flows with different providers.

How It Works

  • Click an OAuth button to open the provider's authorization page
  • The application intercepts callback URLs (like http://localhost:*)
  • Authorization codes are extracted and displayed
  • Popup windows are automatically closed after processing

Configuration

To test with real OAuth providers:

  1. Replace YOUR_CLIENT_ID with your actual client IDs
  2. Configure callback URLs in your OAuth provider settings
  3. Update the redirect_uri parameters as needed
